Section 2.1 Sets

Subsection 2.1.1 Set Basics

Definition 2.1.1.

A set is a collection of objects called elements

Example 2.1.2.

Write the set of letters in the phrase "The University of Oklahoma"

Solution.

{T,h,e,u,n,i,v,r,s,y,o,k,l,a,h,m}

Example 2.1.3.

Write out the set of classes you're taking right now.

Solution.

Answers vary

Definition 2.1.4.

A set \(A\) is said to contain the set \(B\) if all elements of \(B\) are also elements of \(A\text{.}\)

If the set \(A\) contains the set \(B\) but is not equal to \(B\text{,}\) we write \(B\subset A\text{.}\) If \(A\) contains \(B\) and the two sets are equal (that is, if they contain the exact same elements), then we write \(B\subseteq A\text{.}\)

Definition 2.1.5.

If \(B\subset A\text{,}\) then we say that \(B\) is a subset of \(A\text{.}\) If \(B\) has fewer elements than \(A\text{,}\) then we say \(B\) is a proper subset of \(A\text{.}\)

Example 2.1.6.

Let \(A =\) {Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday}.

  1. If \(B\) is the subset of weekend days, write out \(B\text{.}\)

  2. \(C\) is the subset of days with the letter n. What is \(C\text{?}\)

Solution.

  1. \(B =\) {Saturday, Sunday}

  2. \(C=\) {Monday, Wednesday, Sunday}

Subsection 2.1.2 Unions, Intersections

Example 2.1.7.

Let \(A=\) {1,2,3,5,7,8}, \(B=\) {2,4,5,6,8,9}.

  1. \(A\) is not a subset of \(B\) and \(B\) is not a subset of \(A\text{.}\) Why?

  2. What elements do \(A\) and \(B\) have in common?

  3. If we combined \(A\) and \(B\text{,}\) what elements would we have?

Solution.

All elements of \(A\) are not elements of \(B\) and vice versa

\(\displaystyle A\cap B = \lrbrace{2,5,8}\)

\(\displaystyle A\cup B = \lrbrace{1,2,3,4,5,6,7,8,9}\)

We have specific terminology for looking for common elements between sets and smashing two sets together.

Definition 2.1.8.

The union of two sets \(A\) and \(B\) is the collection of elements that are contained in either \(A\) or \(B\) or both \(A\) and \(B\text{.}\) We denote the union by writing \(A\cup B\text{.}\)

The intersection of two sets \(A\) and \(B\) is the collection of elements that are contained in both \(A\) and \(B\text{.}\) We denote the intersection by writing \(A\cap B\text{.}\)

Example 2.1.9.

Let \(A=\) {1,2,3,5,7,8}, \(B=\) {2,4,5,6,8,9}.

  1. Write \(A\cup B\text{.}\)

  2. Write \(A\cap B\text{.}\)

Solution.

  1. \(\displaystyle A\cup B = \lrbrace{1,2,3,4,5,6,7,8,9}\)

  2. \(\displaystyle A\cap B = \lrbrace{2,5,8}\)

Example 2.1.10.

Let \(A =\) {even numbers}, \(B=\) {odd numbers}, and \(C={1,2,3,4}\)

  1. What is \(A\cup B\text{?}\)

  2. What is \(A\cap C\text{?}\)

  3. What is \(A\cap B\text{?}\)

Solution.

  1. \(A\cup B =\) {all positive integers}

  2. \(\displaystyle A\cap C = \lrbrace{2,4}\)

  3. \(\displaystyle A\cap B = \emptyset\)

Definition 2.1.11.

The empty set is the set which contains no elements. We denote it by \(\emptyset\text{.}\)

Subsection 2.1.3 Universal Sets

Example 2.1.12.

Consider the set \(X = \){a,e,i,o,u}. If \(X\) is a subset of a larger set \(Y\text{,}\) what could be a description of \(Y\text{?}\)

Solution.

There are many different answers, but lead them to "the set of all letters".

Definition 2.1.13.

The universal set is the set which contains all elements we are interested in.

Example 2.1.14.

Consider the set \(D=\){chihuahua, labrador, malamute, viszla, greyhound}. What is the universal set \(U\) for \(D\text{?}\)

Solution.

There could be some discussion about this, but "the set of dog breeds" is likely most appropriate.

Example 2.1.15.

Let \(C=\){Iowa State, Kansas State, Kansas, Oklahoma, Oklahoma State}.

  1. What could be the universal set \(U\) for \(C\text{?}\)

  2. What elements from the universal set are missing from \(C\text{?}\)

Solution.

  1. It could be "colleges in the US", but we're going for "colleges in the Big 12".

  2. Assuming we get to the "colleges in the Big 12" as the universal set, the missing elements are {Texas, Texas Tech, Baylor, TCU, West Virginia}

Definition 2.1.16.

The complement of a set \(A\) is the collection of elements which are in the universal set \(U\) but not in \(A\text{.}\) We denote the complement of \(A\) by \(A^c\)

Example 2.1.17.

You are given a set \(A\) and its universal set. Identify the complement \(A^c\text{.}\)

  1. \(A=\lrbrace{1,4,6,7,9}\) and \(U =\) {whole numbers from 1 and 10}

  2. \(A=\) {red, yellow, violet} and \(U=\) {colors of the rainbow}

Solution.

  1. \(\displaystyle A^c = \lrbrace{2,3,5,8,10}\)

  2. \(A^c=\) {orange, green, blue, indigo}

Example 2.1.18.

Let \(K = \) {Beetlejuice, The Fifth Element, Pulp Fiction}, \(H = \) {Beetlejuice, Pulp Fiction, Milk, Pi, The Shining}, and \(L = \) {The Fifth Element, Pi, Milk, Anchorman, Friday}

  1. Find \((H\cap K)\cup L\)

  2. Find \(H\cap (K\cup L)\)

  3. Find \((H\cap K)^c\cap L\)

Solution.

  1. \((H\cap K)\cup L=\) {Beetlejuice, Pulp Fiction, The Fifth Element, Pi, Milk, Anchorman, Friday}

  2. Find \(H\cap (K\cup L)=\) {Beetlejuice, Pulp Fiction}

  3. Find \((H\cap K)^c\cap L=\) {Anchorman, Friday}

Subsection 2.1.5 Cardinality

Definition 2.1.23.

The cardinality of a set is the number of elements it has. We will denote the cardinality of the set \(A\) by writing \(n(A)\text{.}\)

Example 2.1.24.

Let \(A\) be the set of students in class today. What is the cardinality of \(A\text{?}\)

Solution.

Answer varies

Example 2.1.25.

The set \(M\) contains the months which end in -ber and \(N\) contains the months which do not.

  1. What is \(n(A)\text{?}\)

  2. What is \(n(A\cap B)\text{?}\)

  3. What is \(n(A\cup B)\) and what does that represent?

Solution.

  1. 4

  2. 0

  3. 12, the number of months in the year

Example 2.1.26.

A group of students was polled by the OU Daily. 150 students were asked if they primarily used Twitter or Instagram. 57 said Twitter, 80 said Instagram, and 6 said neither. How many people use both Twitter and Instagram primarily?

Solution.

7

Rules of Cardinality.

\begin{equation*} n(A\cup B) = n(A) - n(B) - n(A\cap B) \end{equation*}
\begin{equation*} n(A^c) = n(U) - n(A) \end{equation*}

Example 2.1.27.

A survey of new drivers showed that in the past month, 30 new drivers learned on automatic transmissions, 15 learned on cars with standard transmissions, and 8 learned on cars of both types. If everyone in the survey learned with one of these transmissions, how many new drivers participated in the survey?

Solution.

37

How many people watched RPDR only?
